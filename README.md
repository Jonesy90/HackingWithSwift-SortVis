# Hacking With Swift - SortVis

## Description
A HWS (Hacking With Swift) object designed to visually show how different sorting algorithms work step by step.
The 3 algorithms being displayed are:
1. Bubble Sort
2. Insertion Sort
3. Quicksort

## What I Learnt
- The three types of basic sorting algorithm (Bubble Sort, Insertion Sort and Quicksort). Understanding the difference between the three and why some are more efficent than others.
- Using the 'GeometryReader' to provide a customised width and height of the rectangle of each element.
- Using the 'Picker' to provide a full list of all the sort functions available.
- Using the 'Timer' property to watch and action upon every time it fires.
- Using 'withAnimation' to provide default animation when the array is sorting at each step.
